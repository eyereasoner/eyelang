Eyelog Playground

Edit an Eyelog program, run it entirely in your browser, load examples or URLs, autosave locally, and share a compact link.

version 0.5.18

This page is open as a local file:// URL. Browser security rules usually block the WebAssembly files in that mode. Run make browser, then serve this directory with python3 -m http.server 8000 and open http://localhost:8000/playground.html.

Input program

Idle.

The playground concatenates loaded background programs before the editor contents. Output without a query is the sorted set of derived triple/3 facts, matching the CLI default.
